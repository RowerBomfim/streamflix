# 🎬 Streamflix — Front-End

> Recriação da interface da Netflix desenvolvida com HTML, CSS e JavaScript puro.

---

## 📋 Sobre o Projeto

Este projeto é um clone da interface da Netflix, criado com o objetivo de praticar e consolidar habilidades de desenvolvimento Front-End. A aplicação simula a experiência de uso da plataforma real, com seleção de perfis, catálogo de conteúdo e alternância entre modo claro e escuro.

---

## ✨ Funcionalidades

- 👤 **Seleção de perfis** — tela inicial com múltiplos perfis de usuário, assim como na Netflix real
- 🎞️ **Catálogo de conteúdo** — página de catálogo carregada de acordo com o perfil selecionado
- 🌙 **Modo escuro / claro** — botão para alternar o tema da interface
- ♿ **Acessibilidade** — uso de atributos `aria-label` e `alt` nas imagens para leitores de tela

## 📁 Estrutura do Projeto

```
streamflix/
│
├── assets/          # Imagens dos perfis e demais recursos visuais
├── catalogo/        # Página de catálogo de conteúdo
├── index.html       # Tela de seleção de perfis
├── style.css        # Estilos da aplicação
└── script.js        # Lógica de interação e alternância de tema
```

---

## 🚀 Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/RowerBomfim/streamflix.git
```

2. Acesse a pasta do projeto:
```bash
cd streamflix
```

3. Abra o arquivo `index.html` no seu navegador.

> Não é necessário instalar nada — o projeto roda direto no navegador!
